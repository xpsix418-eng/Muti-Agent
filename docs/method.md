# Method

This project starts with modular baselines for simulated multi-agent cooperative control:

- Rule-based nearest-intruder pursuit.
- Hungarian assignment for defender-to-intruder pairing.
- MAPPO scaffold with centralized critic and decentralized actors.
- GNN-MAPPO scaffold for relational observations.
- Hierarchical MARL scaffold for option-level coordination.

All methods operate on synthetic simulation states only.
